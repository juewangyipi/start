# Game v0.1

网页单机 · 俯视像素 · 开局可玩垂直切片。

依据仓库根目录 `第一版架构_v0.1.txt`。

## 技术

- Vite + TypeScript + Canvas 2D
- 本地存档：`localStorage`（键 `game-v0.1-save`）

## 开发

```bash
cd v0.1
npm install
npm run dev
```

## 四图

```
村落 ── 草地 ── 河边 ── 森林
```

| 地图 | 内容 |
|------|------|
| 村落 | 商店、仓库、存档点、出生 |
| 草地 | 过渡 · 1 棵树 |
| 河边 | 鱼点 |
| 森林 | 树 + 史莱姆（接触掉血） |

## 操作

| 键 | 作用 |
|----|------|
| WASD / 方向键 | 移动 / 切屏 |
| E / Space | 设施点按 · 采集**按住** |
| B / I | 背包 |
| Esc | 关闭面板 |

## 建议体验路径

1. 向右走到河边/森林采集  
2. 回村落 **E 开商店** 卖货、扩背包  
3. **E 开仓库** 囤货  
4. **E 存档点** 回满血并设重生点  
5. 森林被史莱姆打死 → 回存档点  
6. 关页再开 → 读档；若离开前有采集，会离线补一点资源
